# Social-API-Network
This is the backend for an e-commerce site built with Node.js, Express.js, and MongoDB. It provides APIs for managing products, orders, and users

## Installation

To get started, clone this repository to your local machine and run the following commands:

sql
npm install
npm start

This will install all the dependencies and start the server
